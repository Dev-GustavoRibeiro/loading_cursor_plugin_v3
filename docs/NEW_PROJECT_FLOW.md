# New Project Flow

## Step 1
Use `project-bootstrap`.

## Step 2
Answer discovery questions:
- business objective
- target audience
- pages/modules
- brand/palette
- references
- stack constraints
- integrations
- timeline

## Step 3
Approve:
- palette
- style contract
- architecture

## Step 4
Start implementation with the specialist agent that best fits the project.

## Step 5
Run Playwright validation and final design/security/performance audits.
