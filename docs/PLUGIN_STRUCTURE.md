# Plugin Structure

- `.cursor-plugin/plugin.json` → plugin manifest
- `.cursor-plugin/marketplace.json` → optional marketplace manifest for the repo
- `.mcp.json` → packaged MCP server definitions
- `.cursor/rules/` → persistent rules
- `.cursor/agents/` → specialist agents
- `.cursor/skills/` → reusable skills
- `config/` → palette + style contracts
- `templates/` → reusable intake and RFC files
- `mcp-presets/` → alternative MCP bundles
- `scripts/doctor/` → runtime checks
- `scripts/bootstrap/` → project setup helpers
