Use the project-bootstrap skill. Ask me the discovery questions, then create the brief, page map, architecture RFC, and lock the palette in JSON.

## Landing page
Act as the landing-page-conversion-architect. Build the section strategy first, then propose the design system, then the implementation plan.

Act as the supabase-platform-specialist and frontend-architecture-lead together. Define the schema, auth, RLS, design system, and release plan before coding.

Use the figma-handoff skill. Pull structure from the design, map components, and propose the implementation plan.

Act as the observability-debugging-specialist. Use Sentry and Playwright where appropriate, identify root causes, and propose the safest fix path.
