# Playwright QA

Use this skill to validate critical flows.

## Validate
- layout loads correctly
- CTA paths work
- forms validate properly
- navigation works
- responsive behavior is acceptable
- obvious regressions are caught

## Output
Create an actionable QA summary and propose fixes.
