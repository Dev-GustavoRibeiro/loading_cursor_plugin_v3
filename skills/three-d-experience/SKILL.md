# 3D Experience

Use this skill when the site may benefit from 3D.

## Evaluate first
- does 3D improve the story?
- is the target hardware/browser likely to handle it?
- can motion remain performant and tasteful?

## Deliver
- a 3D usage rationale
- section placement
- performance constraints
- fallback states
