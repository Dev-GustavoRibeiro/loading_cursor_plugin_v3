# Design to Code

Use this skill when implementing from design references.

## Workflow
1. confirm palette contract
2. inspect design source or references
3. identify layout primitives, spacing patterns, surfaces, and components
4. map them into reusable code components
5. validate the result with Playwright when possible

## Goal
Translate design intent into maintainable code, not a brittle one-off screen.
