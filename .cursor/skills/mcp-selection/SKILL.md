# MCP Selection

Use this skill to choose which MCP servers should be active for a given task.

## Selection guide
- current framework/library docs → Context7
- browser validation / E2E / interaction debugging → Playwright
- Supabase project work → Supabase
- design-aware implementation → Figma
- issue tracking / product operations → Linear
- production debugging / observability → Sentry
- repo automation / PR workflows → GitHub

## Safety
- use the minimum useful set
- avoid write-capable production operations by default
- prefer dev/staging for infrastructure changes
