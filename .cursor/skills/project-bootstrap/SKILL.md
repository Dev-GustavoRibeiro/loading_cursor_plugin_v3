# Project Bootstrap

Use this skill when starting a new project or re-initializing an existing one.

## Objectives
- run discovery
- confirm visual direction
- lock the approved palette
- create the first repository artifacts
- propose the initial MCP stack
- recommend the correct specialist agent for implementation

## Required outputs
- `docs/project-brief.md`
- `config/project-palette.json`
- `config/project-style-contract.json`
- `docs/page-map.md`
- `docs/architecture-rfc.md`

## Rules
- ask for or confirm the palette first
- do not generate the UI system before palette approval
- if the project belongs to Loading, you may propose Loading defaults but still ask for confirmation
